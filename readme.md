## Overview

This repository hosts the notebooks and in-class codebase for the lectures in CS 1053.

## Setup

0. Make sure you have [uv](https://docs.astral.sh/uv/getting-started/installation/) installed.

1. Clone this repository.

2. Launch JupyterLab:

   ```
   uv run jupyter lab
   ```

   and use the JupyterLab interface to navigate around among notebooks and code files.

## Contents

* `src/` - Source files for the course codebase
* `notebooks/` - Per-class notebooks
